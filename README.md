# Análise de Dados - Embraer

## Descrição

Projeto de análise de dados desenvolvido em Python, com uso de notebooks e scripts para processamento, limpeza, visualização e interpretação de dados relacionados à Embraer.

## Tecnologias Utilizadas

- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Funcionalidades

- Manipulação e análise exploratória de dados
- Visualização gráfica
- Geração de relatórios analíticos

## Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/vitoriasaldanha22/Analise-de-Dados-Embraer.git
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute os notebooks (.ipynb) ou scripts Python conforme a atividade desejada.
